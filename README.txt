** El proyecto representa la simulación del saque de voleibol.
** Utiliza tres objetos 3D: mano, balón y red.
** Actualmente al ejecutar el programa:
    - se abre una ventana sobre la cual se pueden visualizar 
      los tres objetos en formato 2D.
    - se puede interactuar con el balón al utilizar las flechas del teclado
      ARRIBA/ABAJO. Con estas teclas el balón se mueve hacia arriba o hacia 
      abajo para obtener una altura deseada.
    - con la tecla ENTER el balón inicia su desplazamiento sobre una trayectoria.
    - con la tecla SUPRIMIR (o DELETE) el balón regresa a su posición inicial.
    - con la tecla ESC se puede cerrar la ventana.
